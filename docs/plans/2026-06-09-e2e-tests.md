# Plan: E2E Test Suite + cargo-rail CI

**Design doc:** same file (brainstorm output above is incorporated here)

## Goal

Close the binary-level integration gap for `crs` and add a scenario/pipeline e2e harness,
then wire CI via cargo-rail for smart change-detection test selection.

## Architecture

- **Crates affected:** `crates/crs` (Tier 1 binary tests), new `crates/e2e` (Tier 2+3), root
  `Cargo.toml` (new member), `.config/rail.toml` (new)
- **New types:** `Scenario` struct (TOML-deserialized), `BinHarness` helper (e2e common)
- **Data flow:**
  - Tier 1: test spawns `crs` binary → pipes JSON payload → asserts stdout/exit
  - Tier 2: runner loads `fixtures/scenarios/*.toml` → drives binaries → asserts each phase
  - Tier 3: test builds payloads inline → threads through all 4 hooks → asserts final state

## Tech Stack

- Rust 2024 edition, existing workspace deps (`tempfile`, `serde_json`, `toml`, `serde`)
- No new dependencies
- `cargo-rail` for CI change-detection (`cargo rail run --merge-base`)

### Task 4: Tier 1 — `crs filter` binary tests

**Crate:** `crs`
**File(s):** `crates/crs/tests/filter_binary.rs`
**Run:** `cargo nextest run -p crs`

The payload format for `crs filter` (PostToolUse hook):

```json
{
  "tool_name": "Bash",
  "tool_input": { "command": "<cmd>" },
  "tool_response": { "output": "<text>", "exit_code": 0 }
}
```

`CRS_FILTERS` env var points to a temp TOML file with the filter config for the test.

Output format on suppress: `{"type":"result","message":"","decision":"allow"}`
Output format on replace: `{"type":"result","message":"<text>","decision":"allow"}`
No stdout on passthrough.

1. Write failing test — create `crates/crs/tests/filter_binary.rs`:

   ```rust
   #[path = "common_bin.rs"]
   mod common_bin;

   use common_bin::run_crs;
   use std::io::Write;
   use tempfile::NamedTempFile;

   fn filter_toml(pattern: &str, mode: &str) -> NamedTempFile {
       let mut f = NamedTempFile::new().unwrap();
       write!(
           f,
           "[[filters]]\npattern = {pattern:?}\nmode = {mode:?}\n"
       )
       .unwrap();
       f
   }

   fn post_payload(cmd: &str, output: &str, exit_code: i32) -> String {
       format!(
           r#"{{"tool_name":"Bash","tool_input":{{"command":{cmd:?}}},"tool_response":{{"output":{output:?},"exit_code":{exit_code}}}}}"#
       )
   }

   #[test]
   fn filter_binary_suppress_on_exit_zero() {
       let cfg = filter_toml("cargo test", "failures-only");
       let payload = post_payload("cargo test --release", "all passed", 0);
       let out = run_crs(
           "filter",
           &payload,
           &[("CRS_FILTERS", cfg.path().to_str().unwrap())],
       );
       assert!(out.status.success(), "exit: {:?}", out.status);
       let stdout = String::from_utf8_lossy(&out.stdout);
       assert!(
           stdout.contains(r#""message":""#),
           "expected suppress (empty message), got: {stdout}"
       );
   }

   #[test]
   fn filter_binary_passthrough_on_no_matching_rule() {
       let cfg = filter_toml("cargo nextest", "failures-only");
       let payload = post_payload("doob todo list", "some output", 0);
       let out = run_crs(
           "filter",
           &payload,
           &[("CRS_FILTERS", cfg.path().to_str().unwrap())],
       );
       assert!(out.status.success(), "exit: {:?}", out.status);
       // Passthrough: no emit_message call, so stdout should be empty
       let stdout = String::from_utf8_lossy(&out.stdout);
       assert!(
           stdout.is_empty() || !stdout.contains("message"),
           "expected silent passthrough, got: {stdout}"
       );
   }

   #[test]
   fn filter_binary_replace_on_truncate() {
       let mut f = NamedTempFile::new().unwrap();
       write!(
           f,
           "[[filters]]\npattern = \"doob\"\nmode = \"truncate\"\nmax_lines = 2\n"
       )
       .unwrap();
       let long = "line1\nline2\nline3\nline4\nline5";
       let payload = post_payload("doob todo list", long, 0);
       let out = run_crs(
           "filter",
           &payload,
           &[("CRS_FILTERS", f.path().to_str().unwrap())],
       );
       assert!(out.status.success());
       let stdout = String::from_utf8_lossy(&out.stdout);
       assert!(
           stdout.contains("omitted"),
           "expected truncation marker, got: {stdout}"
       );
   }

   #[test]
   fn filter_binary_non_bash_tool_is_silent() {
       let cfg = filter_toml("cargo test", "failures-only");
       let payload = r#"{"tool_name":"Read","tool_input":{"command":"cargo test"},"tool_response":{"output":"ok","exit_code":0}}"#;
       let out = run_crs(
           "filter",
           payload,
           &[("CRS_FILTERS", cfg.path().to_str().unwrap())],
       );
       assert!(out.status.success());
       let stdout = String::from_utf8_lossy(&out.stdout);
       // Non-Bash tool: cmd_filter returns early, no output
       assert!(
           stdout.is_empty() || !stdout.contains("message"),
           "expected silent for non-Bash, got: {stdout}"
       );
   }
   ```

   Run: `cargo nextest run -p crs -- filter_binary`
   Expected: all 4 PASS.

2. Verify: `cargo nextest run -p crs` → all green, `cargo clippy -p crs -- -D warnings` → 0 warnings.

3. Run: `git branch --show-current` — confirm on `main`.

### Task 5: Tier 1 — `crs rewrite` binary tests

**Crate:** `crs`
**File(s):** `crates/crs/tests/rewrite_binary.rs`
**Run:** `cargo nextest run -p crs`

`crs rewrite` reads PreToolUse JSON, exits 1 on no match, exits 0 with JSON on match.

Output format on rewrite:

```json
{
  "hookSpecificOutput": {
    "hookEventName": "PreToolUse",
    "permissionDecision": "allow",
    "permissionDecisionReason": "crs rewrite: <cmd>",
    "updatedInput": { "command": "<cmd>" }
  }
}
```

`CRS_FILTERS` must contain a `[[rewrites]]` section.

1. Write failing test — create `crates/crs/tests/rewrite_binary.rs`:

   ```rust
   #[path = "common_bin.rs"]
   mod common_bin;

   use common_bin::run_crs;
   use std::io::Write;
   use tempfile::NamedTempFile;

   fn rewrite_toml(pattern: &str, replace: &str) -> NamedTempFile {
       let mut f = NamedTempFile::new().unwrap();
       write!(
           f,
           "[[rewrites]]\npattern = {pattern:?}\nreplace = {replace:?}\n"
       )
       .unwrap();
       f
   }

   fn pre_payload(cmd: &str) -> String {
       format!(r#"{{"tool_name":"Bash","tool_input":{{"command":{cmd:?}}}}}"#)
   }

   #[test]
   fn rewrite_binary_matching_rule_exits_zero_with_json() {
       let cfg = rewrite_toml("^cargo test(.*)", "cargo nextest run$1");
       let payload = pre_payload("cargo test --release");
       let out = run_crs(
           "rewrite",
           &payload,
           &[("CRS_FILTERS", cfg.path().to_str().unwrap())],
       );
       assert!(out.status.success(), "exit: {:?}\nstderr: {}", out.status,
               String::from_utf8_lossy(&out.stderr));
       let stdout = String::from_utf8_lossy(&out.stdout);
       assert!(
           stdout.contains("cargo nextest run --release"),
           "expected rewritten command in output, got: {stdout}"
       );
       assert!(
           stdout.contains("updatedInput"),
           "expected hookSpecificOutput JSON, got: {stdout}"
       );
   }

   #[test]
   fn rewrite_binary_no_match_exits_one() {
       let cfg = rewrite_toml("^cargo test(.*)", "cargo nextest run$1");
       let payload = pre_payload("doob todo list");
       let out = run_crs(
           "rewrite",
           &payload,
           &[("CRS_FILTERS", cfg.path().to_str().unwrap())],
       );
       assert!(
           !out.status.success(),
           "expected exit 1 on no match, got: {:?}",
           out.status
       );
       let stdout = String::from_utf8_lossy(&out.stdout);
       assert!(stdout.is_empty(), "expected empty stdout on no match, got: {stdout}");
   }

   #[test]
   fn rewrite_binary_non_bash_exits_one() {
       let cfg = rewrite_toml("^cargo test(.*)", "cargo nextest run$1");
       let payload = r#"{"tool_name":"Read","tool_input":{"command":"cargo test"}}"#;
       let out = run_crs(
           "rewrite",
           payload,
           &[("CRS_FILTERS", cfg.path().to_str().unwrap())],
       );
       assert!(!out.status.success(), "expected exit 1 for non-Bash tool");
   }
   ```

   Run: `cargo nextest run -p crs -- rewrite_binary`
   Expected: all 3 PASS.

   ```rust
   #[path = "common_bin.rs"]
   mod common_bin;

   use common_bin::crs_bin;
   use std::io::Write;
   use std::process::Command;
   use tempfile::NamedTempFile;

   #[test]
   fn validate_binary_valid_rules_exits_zero() {
       let mut f = NamedTempFile::new().unwrap();
       // Minimal valid rules file — empty array is valid.
       write!(f, r#"{{"rules":[]}}"#).unwrap();
       let out = Command::new(crs_bin())
           .arg("validate")
           .env("COURSERS_RULES", f.path())
           .output()
           .unwrap();
       assert!(
           out.status.success(),
           "expected exit 0 with valid rules, got: {:?}\nstderr: {}",
           out.status,
           String::from_utf8_lossy(&out.stderr)
       );
   }

   #[test]
   fn validate_binary_bad_json_exits_nonzero() {
       let mut f = NamedTempFile::new().unwrap();
       write!(f, "not valid json {{{{").unwrap();
       let out = Command::new(crs_bin())
           .arg("validate")
           .env("COURSERS_RULES", f.path())
           .output()
           .unwrap();
       // Bad rules: validate should either exit non-zero or print errors.
       // Accept either: non-zero exit OR stdout containing "error"/"invalid".
       let stdout = String::from_utf8_lossy(&out.stdout);
       let stderr = String::from_utf8_lossy(&out.stderr);
       let flagged = !out.status.success()
           || stdout.to_lowercase().contains("error")
           || stderr.to_lowercase().contains("error");
       assert!(
           flagged,
           "expected validate to flag bad JSON, got exit {:?}\nstdout: {stdout}\nstderr: {stderr}",
           out.status
       );
   }
   ```

2. Write failing test — create `crates/e2e/tests/scenarios.rs`:

   ```rust
   use serde::Deserialize;
   use std::io::Write;
   use std::path::{Path, PathBuf};
   use std::process::{Command, Output, Stdio};
   use tempfile::{NamedTempFile, TempDir};

   // ---------------------------------------------------------------------------
   // Scenario schema
   // ---------------------------------------------------------------------------

   #[derive(Debug, Deserialize)]
   struct ScenarioFile {
       scenario: ScenarioMeta,
       pre: Option<PrePhase>,
       filter: Option<FilterPhase>,
       post_failures: Option<PostFailures>,
   }

   #[derive(Debug, Deserialize)]
   struct ScenarioMeta {
       name: String,
       #[allow(dead_code)]
       description: Option<String>,
       coursers_rules_json: Option<String>,
       crs_filters_toml: Option<String>,
   }

   #[derive(Debug, Deserialize)]
   struct PrePhase {
       command: String,
       expected_verdict: String, // "block" | "allow"
   }

   #[derive(Debug, Deserialize)]
   struct FilterPhase {
       command: String,
       output: String,
       exit_code: i32,
       expected_result: String, // "suppress" | "replace" | "passthrough"
   }

   #[derive(Debug, Deserialize)]
   struct PostFailures {
       command: String,
       count: u32,
   }

   // ---------------------------------------------------------------------------
   // Harness helpers
   // ---------------------------------------------------------------------------

   fn workspace_bin(name: &str) -> PathBuf {
       let manifest = Path::new(env!("CARGO_MANIFEST_DIR"));
       let workspace = manifest.parent().unwrap().parent().unwrap();
       let debug = workspace.join("target/debug").join(name);
       let release = workspace.join("target/release").join(name);
       if release.exists() {
           release
       } else {
           assert!(
               debug.exists(),
               "binary {name:?} not found — run `cargo build --workspace` first\n\
                checked: {}",
               debug.display()
           );
           debug
       }
   }

   fn run_bin(
       bin: &str,
       subcommand: &str,
       payload: &str,
       envs: &[(&str, &str)],
   ) -> Output {
       let mut cmd = Command::new(workspace_bin(bin));
       cmd.arg(subcommand)
           .stdin(Stdio::piped())
           .stdout(Stdio::piped())
           .stderr(Stdio::piped());
       for (k, v) in envs {
           cmd.env(k, v);
       }
       let mut child = cmd.spawn().unwrap_or_else(|e| panic!("spawn {bin} {subcommand}: {e}"));
       child
           .stdin
           .as_mut()
           .unwrap()
           .write_all(payload.as_bytes())
           .unwrap();
       child.wait_with_output().unwrap()
   }

   fn pre_payload(cmd: &str) -> String {
       format!(r#"{{"tool_name":"Bash","tool_input":{{"command":{cmd:?}}}}}"#)
   }

   fn post_payload(cmd: &str, exit_code: i32) -> String {
       format!(
           r#"{{"tool_name":"Bash","tool_input":{{"command":{cmd:?}}},"tool_response":{{"exit_code":{exit_code}}}}}"#
       )
   }

   fn post_filter_payload(cmd: &str, output: &str, exit_code: i32) -> String {
       format!(
           r#"{{"tool_name":"Bash","tool_input":{{"command":{cmd:?}}},"tool_response":{{"output":{output:?},"exit_code":{exit_code}}}}}"#
       )
   }

   // ---------------------------------------------------------------------------
   // Runner
   // ---------------------------------------------------------------------------

   fn run_scenario(path: &Path) {
       let content = std::fs::read_to_string(path)
           .unwrap_or_else(|e| panic!("read {}: {e}", path.display()));
       let sf: ScenarioFile = toml::from_str(&content)
           .unwrap_or_else(|e| panic!("parse {}: {e}", path.display()));
       let name = &sf.scenario.name;

       let tmp = TempDir::new().unwrap();
       let state_path = tmp.path().join("state.json");

       // Write rules file
       let rules_content = sf
           .scenario
           .coursers_rules_json
           .as_deref()
           .unwrap_or(r#"{"rules":[]}"#);
       let mut rules_file = NamedTempFile::new().unwrap();
       write!(rules_file, "{rules_content}").unwrap();

       // Write CRS_FILTERS file (if provided)
       let filters_file = sf.scenario.crs_filters_toml.as_ref().map(|toml| {
           let mut f = NamedTempFile::new().unwrap();
           write!(f, "{toml}").unwrap();
           f
       });

       let envs_pre: Vec<(&str, String)> = vec![
           ("COURSERS_RULES", rules_file.path().to_string_lossy().into_owned()),
           ("COURSERS_STATE", state_path.to_string_lossy().into_owned()),
       ];
       let envs_pre: Vec<(&str, &str)> = envs_pre
           .iter()
           .map(|(k, v)| (*k, v.as_str()))
           .collect();

       // Phase: post_failures (record failures before testing pre-block)
       if let Some(ref pf) = sf.post_failures {
           for _ in 0..pf.count {
               let payload = post_payload(&pf.command, 1);
               run_bin("coursers", "post", &payload, &envs_pre);
           }
       }

       // Phase: pre
       if let Some(ref pre) = sf.pre {
           let payload = pre_payload(&pre.command);
           let out = run_bin("coursers", "pre", &payload, &envs_pre);
           match pre.expected_verdict.as_str() {
               "block" => assert!(
                   !out.status.success(),
                   "[{name}] pre: expected block (non-zero exit), got {:?}",
                   out.status
               ),
               "allow" => assert!(
                   out.status.success(),
                   "[{name}] pre: expected allow (exit 0), got {:?}\nstderr: {}",
                   out.status,
                   String::from_utf8_lossy(&out.stderr)
               ),
               other => panic!("[{name}] unknown expected_verdict: {other:?}"),
           }
       }

       // Phase: filter
       if let Some(ref fil) = sf.filter {
           let mut filter_envs: Vec<(&str, String)> = vec![];
           if let Some(ref f) = filters_file {
               filter_envs.push(("CRS_FILTERS", f.path().to_string_lossy().into_owned()));
           }
           let filter_envs: Vec<(&str, &str)> =
               filter_envs.iter().map(|(k, v)| (*k, v.as_str())).collect();
   ```

### Task 8: Tier 3 — Pipeline harness

**Crate:** `e2e`
**File(s):** `crates/e2e/tests/pipeline.rs`
**Run:** `cargo build --workspace && cargo nextest run -p e2e -- pipeline`

The pipeline sequence per test:

1. `crs rewrite` (PreToolUse) → maybe-rewritten command
2. `coursers pre` (PreToolUse) → verdict
3. `coursers post` (PostToolUse) → side-effect on state file (if allowed)
4. `crs filter` (PostToolUse) → final output

5. Write failing test — create `crates/e2e/tests/pipeline.rs`:

   ```rust
   use std::io::Write;
   use std::path::{Path, PathBuf};
   use std::process::{Command, Output, Stdio};
   use tempfile::{NamedTempFile, TempDir};

   fn workspace_bin(name: &str) -> PathBuf {
       let manifest = Path::new(env!("CARGO_MANIFEST_DIR"));
       let workspace = manifest.parent().unwrap().parent().unwrap();
       let debug = workspace.join("target/debug").join(name);
       let release = workspace.join("target/release").join(name);
       if release.exists() { release } else { debug }
   }

   fn run_bin(bin: &str, sub: &str, payload: &str, envs: &[(&str, &str)]) -> Output {
       let mut cmd = Command::new(workspace_bin(bin));
       cmd.arg(sub)
           .stdin(Stdio::piped())
           .stdout(Stdio::piped())
           .stderr(Stdio::piped());
       for (k, v) in envs {
           cmd.env(k, v);
       }
       let mut child = cmd.spawn().unwrap_or_else(|e| panic!("spawn {bin} {sub}: {e}"));
       child.stdin.as_mut().unwrap().write_all(payload.as_bytes()).unwrap();
       child.wait_with_output().unwrap()
   }

   struct Pipeline {
       state: TempDir,
       rules: NamedTempFile,
       filters: NamedTempFile,
   }

   impl Pipeline {
       fn new(rules_json: &str, filters_toml: &str) -> Self {
           let state = TempDir::new().unwrap();
           let mut rules = NamedTempFile::new().unwrap();
           write!(rules, "{rules_json}").unwrap();
           let mut filters = NamedTempFile::new().unwrap();
           write!(filters, "{filters_toml}").unwrap();
           Pipeline { state, rules, filters }
       }

       fn pre_envs(&self) -> [(&str, String); 2] {
           [
               ("COURSERS_RULES", self.rules.path().to_string_lossy().into_owned()),
               ("COURSERS_STATE", self.state.path().join("state.json").to_string_lossy().into_owned()),
           ]
       }

       fn filter_envs(&self) -> [(&str, String); 1] {
           [("CRS_FILTERS", self.filters.path().to_string_lossy().into_owned())]
       }

       fn run_pre(&self, command: &str) -> Output {
           let payload = format!(r#"{{"tool_name":"Bash","tool_input":{{"command":{command:?}}}}}"#);
           let envs = self.pre_envs();
           let envs: Vec<(&str, &str)> = envs.iter().map(|(k, v)| (*k, v.as_str())).collect();
           run_bin("coursers", "pre", &payload, &envs)
       }

       fn run_post(&self, command: &str, exit_code: i32) -> Output {
           let payload = format!(
               r#"{{"tool_name":"Bash","tool_input":{{"command":{command:?}}},"tool_response":{{"exit_code":{exit_code}}}}}"#
           );
           let envs = self.pre_envs();
           let envs: Vec<(&str, &str)> = envs.iter().map(|(k, v)| (*k, v.as_str())).collect();
           run_bin("coursers", "post", &payload, &envs)
       }

       fn run_filter(&self, command: &str, output: &str, exit_code: i32) -> Output {
           let payload = format!(
               r#"{{"tool_name":"Bash","tool_input":{{"command":{command:?}}},"tool_response":{{"output":{output:?},"exit_code":{exit_code}}}}}"#
           );
           let envs = self.filter_envs();
           let envs: Vec<(&str, &str)> = envs.iter().map(|(k, v)| (*k, v.as_str())).collect();
           run_bin("crs", "filter", &payload, &envs)
       }

       fn run_rewrite(&self, command: &str) -> Option<String> {
           let payload = format!(r#"{{"tool_name":"Bash","tool_input":{{"command":{command:?}}}}}"#);
           let envs = self.filter_envs();
           let envs: Vec<(&str, &str)> = envs.iter().map(|(k, v)| (*k, v.as_str())).collect();
           let out = run_bin("crs", "rewrite", &payload, &envs);
           if !out.status.success() {
               return None;
           }
           let stdout = String::from_utf8_lossy(&out.stdout);
           let v: serde_json::Value = serde_json::from_str(&stdout).ok()?;
           v["hookSpecificOutput"]["updatedInput"]["command"]
               .as_str()
               .map(str::to_string)
       }
   }

   // ---------------------------------------------------------------------------
   // Pipeline tests
   // ---------------------------------------------------------------------------

   #[test]
   fn pipeline_grep_blocked() {
       let pipe = Pipeline::new(
           r#"{"rules":[{"id":"no-grep","pattern":"\\bgrep\\b","action":"block",
              "message":"Use Grep tool","enabled":true,"exceptions":[]}]}"#,

   }

   #[test]
   fn pipeline_ls_allowed_and_filter_suppresses() {
       let pipe = Pipeline::new(
           r#"{"rules":[]}"#,
           "[[filters]]\npattern = \"ls\"\nmode = \"failures-only\"\n",
       );
       // pre: allowed
       let pre = pipe.run_pre("ls -la");
       assert!(pre.status.success(), "expected allow, got {:?}", pre.status);
       // post: exit 0 → no state written
       pipe.run_post("ls -la", 0);
       assert!(
           !pipe.state.path().join("state.json").exists(),
           "exit-0 post should not write state"
       );
       // filter: suppress on exit 0
       let fil = pipe.run_filter("ls -la", "total 8", 0);
       assert!(fil.status.success());
       let stdout = String::from_utf8_lossy(&fil.stdout);
       assert!(
           stdout.contains(r#""message":""#),
           "expected suppress, got: {stdout}"
       );
   }

   #[test]
   fn pipeline_learned_failure_blocks_after_threshold() {
       let pipe = Pipeline::new(r#"{"rules":[]}"#, "");
       let cmd = "e2e-pipeline-unique-zzz999";
       // Record 3 failures via post
       for _ in 0..3 {
           pipe.run_post(cmd, 1);
       }
       // pre should now block
       let pre = pipe.run_pre(cmd);
       assert!(
           !pre.status.success(),
           "expected block after 3 failures, got {:?}",
           pre.status
       );
   }

   #[test]
   fn pipeline_cargo_test_rewritten_and_filtered() {
       let pipe = Pipeline::new(
           r#"{"rules":[]}"#,
           "[[rewrites]]\npattern = \"^cargo test(.*)\"\nreplace = \"cargo nextest run$1\"\n\
            [[filters]]\npattern = \"cargo nextest\"\nmode = \"failures-only\"\n",
       );
       // rewrite: cargo test → cargo nextest run
       let rewritten = pipe.run_rewrite("cargo test --release");
       assert_eq!(
           rewritten.as_deref(),

           stdout.contains(r#""message":""#),
           "expected suppress on success, got: {stdout}"
       );
   }
   ```
